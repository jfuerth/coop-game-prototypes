Trampoline Game Prototype
FPS: 0
Gamepad: Not connected
Keyboard: Arrow keys or WASD to move • Space to jump • E = Toggle Editor
T = Add trampoline • P = Add platform • R = Remove last platform
Gamepad: Left stick/D-pad to move • A/B/D-pad up to jump • Select = Toggle Editor
X = Add trampoline • Y = Add platform • Bumpers = Remove platform
Level Editor: Click palette tools then click on game area to paint objects
